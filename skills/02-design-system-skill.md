# 02 — Design System Skill

## Purpose

Keep every page visually consistent with the premium pastel dashboard theme.

## Visual Direction

The app must feel:

- Premium
- Soft
- Clean
- Light
- Chart-heavy
- Pastel
- Rounded
- Professional
- Slightly playful with subtle sparkles

## Color Semantics

| Meaning | Color |
|---|---|
| Primary / Active | Purple / Lavender |
| Profit / Success / Safe | Mint / Teal |
| Neutral Info | Sky Blue |
| Warning / Medium Risk | Soft Orange |
| Loss / Danger / Rule Break | Soft Coral / Red |
| Premium Accent | Pink / Violet |

## Card Style

```text
border-radius: 18px to 24px
background: white or soft pastel gradient
border: 1px solid rgba(124, 58, 237, 0.10)
box-shadow: 0 10px 30px rgba(31, 41, 55, 0.06)
```

## Avoid

- Harsh red
- Dark dashboard theme
- Neon colors
- Heavy black shadows
- Random colors
- Square cards
- Off-brand gradients

## QA

- Purple remains primary accent
- Positive states are mint/teal
- Negative states are soft coral/red
- Theme does not drift from reference images
