# 07 — QA Visual Matching Skill

## Purpose

Run this checklist before finishing any page.

## Layout QA

- Sidebar unchanged
- Active sidebar item correct
- Top tabs in correct order
- Active tab correct
- Card count matches reference
- Main chart areas match reference
- Spacing consistent
- No unrelated sections added

## Visual QA

- Theme is pastel premium
- Purple primary accent
- Green/mint positive values
- Coral/red negative values
- Orange warning
- Cards rounded
- Shadows soft
- Background light
- Typography readable at 100% zoom

## Chart QA

- Recharts used
- ResponsiveContainer used
- Tooltips readable
- Axis labels not too big
- Charts do not overflow
- Color semantics correct

## Action QA

- Buttons work
- Modals/drawers open
- Filters update UI
- Export/import mock works
- Destructive actions confirm first
- No fake backend actions

## Responsive QA

- Desktop matches reference
- Tablet usable
- Mobile stacks vertically
- Tabs scroll if needed
- No horizontal overflow
