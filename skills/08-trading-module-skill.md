# 08 — Trading Module Skill

## Purpose

Build Trading as the most detailed module of the app, supporting the user's goal of becoming a full-time trader.

## Trading Tabs

```text
Overview
Funding
Journal
Analytics
Strategy
Backtest
Risk
Review
```

## Overview

Show:

- Total capital
- Monthly P/L
- Win Rate
- Profit Factor
- Drawdown
- Active challenges
- Best edge
- Worst risk
- Trading readiness

## Funding

Show:

- Prop firm accounts
- CFD / Futures type
- Challenge fee
- Reset fee
- Platform fee
- Payout method
- Rules
- Daily loss remaining
- Max drawdown remaining
- Phase progress

## Journal

Show:

- Trades
- Entry / Exit
- SL / TP
- R
- MFE / MAE
- Strategy
- Session
- Emotion
- Mistake
- Screenshot placeholder

## Analytics

Show:

- Equity curve
- Drawdown
- Win/loss distribution
- Asset performance
- Session performance
- MFE / MAE
- Mistake frequency

## Strategy

Show:

- Playbook
- Setup library
- Entry rules
- Invalid conditions
- Backtest result
- Live result
- Strategy comparison

## Backtest

Show:

- Backtest sessions
- FX Replay-style equity curve
- Strategy comparison
- Prop firm simulation
- MFE / MAE scatter
- Time heatmap
- Trade log
- Next test recommendation

## Risk

Show:

- Position sizing
- Risk per trade
- Drawdown buffer
- Exposure
- Correlated risk
- Compounding
- Bot / automation status

## Review

Show:

- Weekly review
- Monthly review
- Mistake review
- Psychology review
- Action plan
- Full-time trader roadmap

## QA

Trading must be graph-heavy, premium, and action-ready.
Do not reduce Trading to a simple journal.
